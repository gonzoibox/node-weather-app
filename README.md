# Node Weather App

A simple weather application built with Node.js and Express. This app allows users to get weather information based on their location.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/gonzoibox/node-weather-app.git
   cd node-weather-app
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To run the application, use the following command:

```bash
node weather-app/app.js "Your Location"
```

Replace "Your Location" with the desired address to get the weather information.

## Features

- Get current weather information based on location.
- User-friendly interface with error handling.
